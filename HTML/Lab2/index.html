คำถาม

  1. ค้นหาข้อมูลว่าแท็ก <!doctype> มีหน้าที่อะไร?
  2. ถ้าอยากให้มี tooltip ใน tag นั้นต้องใช้ attribute ใด?
  3. แท็ก <be>, <ie> ต่างจาก <stronge>, <eme> หรือไม่ อย่างไร?
  4. หา tag ที่สามารถแสดงข้อความกลับหลังได้ เช่น โปรแกรมเมอร์ แสดงเป็น ร์อมเมรกแรปโ

คำตอบ

  1. Doctype ย่อมาจากเอกสารประเภทประกาศ จะแจ้งเว็บเบราว์เซอร์เกี่ยวกับประเภทและเวอร์ชันของ HTML ที่ใช้ในการสร้างเอกสารเว็บ ซึ่งจะช่วยให้เบราว์เซอร์จัดการและโหลดได้อย่างถูกต้อง
  2. title แอตทริบิวต์กำหนดบางข้อมูลเพิ่มเติมเกี่ยวกับองค์ประกอบ
  3. แท็ก <b> ใช้กำหนดข้อความเป็นตัวหนาโดยไม่มีความสำคัญเพิ่มเติม
    แท็ก <strong> ใช้กำหนดข้อความที่มีความสำคัญมาก โดยทั่วไปเนื้อหาภายในจะแสดงเป็นตัวหนา
    แท็ก <i> ใช้กำหนดส่วนหนึ่งของข้อความในเสียงหรืออารมณ์อื่น โดยทั่วไปเนื้อหาภายในจะแสดงเป็นตัวเอียง(มักจะถูกใช้ในการระบุระยะทางเทคนิควลีจากภาษาอื่นคิดชื่อเรือ ฯลฯ)
    แท็ก <em> ใช้กำหนดข้อความที่เน้น โดยทั่วไปเนื้อหาภายในจะแสดงเป็นตัวเอียง(โดยมักจะใช้กับคำพูดเน้นเสียง)
  4. ใช้ แท็ก <bdo> ในการแสดง "โปรแกรมเมอร์" เป็น โปรแกรมเมอร์
